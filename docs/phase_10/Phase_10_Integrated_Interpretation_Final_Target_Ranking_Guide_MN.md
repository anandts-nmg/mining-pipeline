Phase 10 — Integrated Interpretation and Final Target Ranking
Ажлыг хийх дэлгэрэнгүй заавар
Энэ үе шатны зорилго нь өмнөх бүх хайгуулын дата, лабораторийн үр дүн, remote sensing, field mapping, CMCS мэдээлэл, ойролцоох орд/илрэл, геологийн зураг, структур, хувирал, геохими зэрэг бүх нотолгоог нэгтгэж, эцсийн хайгуулын target-уудыг эрэмбэлэх, Go / No-Go шийдвэр гаргах, дараагийн шатны ажил буюу trench, geophysics, scout drilling төлөвлөхөд бэлэн болгох юм.
# 1. Ажлын үндсэн зорилго
Phase 10-д дараах ажлуудыг гүйцэтгэнэ.
- Бүх хайгуулын нотолгоог нэг GIS орчинд нэгтгэнэ.
- Дата бүрийн чанарын шалгалт буюу QA/QC хийнэ.
- Геологи, структур, хувирал, геохими, remote sensing, field evidence, CMCS мэдээллийг давхарлан тайлбарлана.
- Боломжит mineral target polygon-уудыг тодорхойлно.
- Target бүрт оноо өгч A / B / C зэрэглэлд ангилна.
- Target бүр дээр тайлбар sheet бэлтгэнэ.
- Go / No-Go шийдвэр гаргана.
- Final GIS map, ranking table, integrated interpretation report гаргана.
# 2. Ашиглах үндсэн input data
Энэ шатанд өмнөх бүх phase-ийн баталгаажсан дата ашиглагдана. Ялангуяа дараах дата заавал орно.
## 2.1 Геологийн дата
Ашиглах материал:
- Геологийн зураг
- Литологийн зураг
- Интрузив, вулканик, тунамал нэгжүүд
- Хувиралтай бүсүүд
- Хагарал, ан цав, lineament
- Структурын хэмжилт
- Field mapping observation
- Rock unit boundary
- Mineralized zone boundary
Хийх ажил:
- Геологийн нэгжүүдийг target area-тай давхарлуулж шалгана.
- Хүдэржилттэй холбоотой байж болох lithology-г ялгана.
- Интрузив биет, контакт бүс, skarn potential, alteration halo, vein zone зэргийг тодорхойлно.
- Ордын загвартай тохирох геологийн нөхцөл байгаа эсэхийг шалгана.
## 2.2 Металлогени ба ойролцоох орд/илрэл
Ашиглах материал:
- CMCS database
- MRAM/АМГТГ-ийн mineral occurrence мэдээлэл
- Ойролцоох орд, илрэл, prospect
- Бүс нутгийн metallogenic map
- Known deposit trend
- Ore district boundary
- Historical exploration result
Хийх ажил:
- Талбайн ойролцоо ямар төрлийн орд, илрэл байгааг шалгана.
- Зэс, алт, мөнгө, полиметалл, молибден, төмөр, никель гэх мэт commodity бүрийг тусад нь бүртгэнэ.
- Ойролцоох орд/илрэлүүд талбайтай нэг structural trend, lithological belt, intrusive belt, alteration corridor дотор байгаа эсэхийг шалгана.
- CMCS мэдээллээс ордын төрөл, эрдсийн бүрдэл, координат, статус, судалгааны түвшинг бүртгэнэ.
- Талбайн target-уудыг ойролцоох орд/илрэлтэй харьцуулж potential resource type тодорхойлно.
Жишээ үнэлгээ:

| Үзүүлэлт | Ач холбогдол |
| --- | --- |
| Ойролцоох Cu-Au порфир орд | Porphyry Cu-Au target магадлал нэмэгдэнэ |
| Au-Ag судлын илрэл | Epithermal эсвэл vein target магадлал нэмэгдэнэ |
| Skarn Cu-Fe илрэл | Интрузив-карбонат контакт бүс чухал болно |
| Polymetallic Pb-Zn-Ag илрэл | Structural vein corridor шалгах шаардлагатай |
| Ni-Cu илрэл | Mafic-ultramafic lithology-г нарийвчлан шалгана |

## 2.3 Stream sediment, heavy mineral дата
Ашиглах материал:
- Stream sediment assay
- Heavy mineral concentrate result
- Drainage basin boundary
- Sample location
- Geochemical anomaly map
- Element association
Хийх ажил:
- Cu, Au, Ag, Pb, Zn, Mo, As, Sb, Bi, W, Sn, Ni, Co зэрэг элементүүдийг тусад нь шалгана.
- Anomalous value-г background, threshold, percentile аргаар тодорхойлно.
- Дээжийн anomalous point-уудыг drainage catchment-тэй холбож source area тодорхойлно.
- Stream anomaly нь target polygon-тай огтлолцож байгаа эсэхийг шалгана.
- Heavy mineral indication байгаа бол эх үүсвэрийн талбайг upstream чиглэлээр тодорхойлно.
## 2.4 pXRF болон лабораторийн assay дата
Ашиглах материал:
- pXRF field reading
- Rock chip assay
- Channel sample assay
- Soil assay
- Lab assay certificate
- QA/QC sample result
- Standard, blank, duplicate result
- pXRF-lab correlation table
Хийх ажил:
- pXRF ба лабораторийн үр дүнг харьцуулна.
- Cu, Zn, Pb, As, Mo, Fe зэрэг pXRF-д сайн хэмжигдэх элементүүдийн correlation шалгана.
- Au зэрэг pXRF-д найдвартай хэмжигдэхгүй элементийг лабораторийн assay дээр үндэслэнэ.
- Duplicate sample-ийн зөрүүг шалгана.
- Blank contamination байгаа эсэхийг шалгана.
- CRM буюу certified reference material зөв хэмжигдсэн эсэхийг баталгаажуулна.
- Outlier value-г шалгаж, алдаа эсвэл жинхэнэ high-grade утга эсэхийг тогтооно.
## 2.5 Remote sensing дата
Ашиглах материал:
- Sentinel imagery
- ASTER alteration index
- KOMPSAT imagery
- ALOS DEM
- Drone imagery
- LiDAR
- Photogrammetry output
- Lineament interpretation
- Alteration map
Хийх ажил:
- ASTER/Sentinel-ээс clay, iron oxide, silica, carbonate alteration indicator гаргана.
- Lineament, fault, fracture zone-уудыг ялгана.
- DEM болон hillshade дээр structural control шалгана.
- Drone/LiDAR дата дээр outcrop, vein, gossan, alteration zone, old working байгаа эсэхийг шалгана.
- Remote sensing anomaly нь field evidence болон geochemical anomaly-той давхцаж байгаа эсэхийг баталгаажуулна.
## 2.6 Field mapping дата
Ашиглах материал:
- Field observation point
- Lithology observation
- Alteration observation
- Mineralization observation
- Structure measurement
- Photo point
- Sample point
- Traverse track
Хийх ажил:
- Хээрийн ажиглалтын цэгүүдийг GIS дээр шалгана.
- Хүдэржилтийн тэмдэглэлүүдийг эрдсийн төрөл, хүч, тархалтаар ангилна.
- Quartz vein, sulfide vein, gossan, malachite, azurite, chalcopyrite, pyrite, galena, sphalerite зэрэг тэмдэглэлүүдийг ялгана.
- Хувирал болон хүдэржилт давхацсан хэсгийг өндөр ач холбогдолтой гэж үзнэ.
- Field photo-г target sheet-д нотлох баримт болгон оруулна.

## Алхам 3. Assay QA/QC шалгах
Энэ алхам нь target ranking хийхээс өмнө заавал дууссан байх ёстой.
### 3.1 Duplicate check
Duplicate sample-ийн үр дүн original sample-тай ойролцоо байгаа эсэхийг шалгана.
Шалгах үзүүлэлт:
- Cu
- Au
- Ag
- Pb
- Zn
- Mo
- As
- Sb
- Ni
- Co
Үнэлгээ:

| Зөрүү | Үнэлгээ |
| --- | --- |
| 0–10% | Маш сайн |
| 10–20% | Зөвшөөрөх боломжтой |
| 20–30% | Анхаарах шаардлагатай |
| >30% | Дахин шалгах шаардлагатай |

### 3.2 Blank check
Blank sample дээр high value гарсан эсэхийг шалгана.
Хэрэв blank дээр Cu, Au, Pb, Zn зэрэг элементүүд өндөр гарвал:
- Contamination байж болзошгүй
- Sample preparation алдаа байж болзошгүй
- Lab batch дахин шалгах шаардлагатай
### 3.3 CRM check
CRM result нь certified value-тэй нийцэж байгаа эсэхийг шалгана.
Хэрэв CRM deviation өндөр бол:
- Тухайн batch-ийн assay result-ийг болгоомжтой ашиглана.
- Lab-аас clarification авна.
- Шаардлагатай бол re-assay санал болгоно.
### 3.4 pXRF-lab correlation
pXRF ба lab result-ийг element бүрээр scatter plot хийж харьцуулна.
Хүчтэй correlation гарвал:
- pXRF дата-г vector trend, anomaly support evidence болгон ашиглаж болно.
Сул correlation гарвал:
- pXRF-г зөвхөн field screening evidence гэж үзнэ.
- Final ranking-д lab assay-г илүү өндөр жинтэй авна.

## Алхам 7. Target polygon үүсгэх
Target polygon үүсгэхдээ зөвхөн нэг төрлийн дата дээр үндэслэхгүй. Доорх нотолгооноос хамгийн багадаа 3 төрлийн evidence давхцсан хэсгийг target болгон авна.
Жишээ:
- Geochemical anomaly + structure + alteration
- Rock assay + field mineralization + remote sensing alteration
- CMCS nearby deposit + favorable lithology + stream sediment anomaly
- Intrusive contact + skarn alteration + Cu/Fe anomaly
- Fault intersection + quartz vein + Au-As-Sb anomaly
Target polygon үүсгэх шалгуур:

| Шалгуур | Тайлбар |
| --- | --- |
| Evidence overlap | 3 ба түүнээс дээш нотолгоо давхцах |
| Geological plausibility | Ордын загвартай нийцэх |
| Assay support | Лабораторийн эсвэл field assay дэмжлэгтэй |
| Structural control | Хагарал, lineament, contact zone-тэй холбоотой |
| Alteration support | Хувирал илэрсэн |
| Size potential | Хайгуулын үргэлжлэл хийх боломжтой хэмжээтэй |
| Accessibility | Талбай дээр очиж шалгах боломжтой |

# 6. Ордын төрлөөр үнэлэх аргачлал
## 6.1 Porphyry Cu-Au target
Өндөр ач холбогдолтой болох нөхцөл:
- Интрузив биеттэй холбоотой
- Cu-Mo-Au anomaly илэрсэн
- Potassic / phyllic / argillic alteration байж болзошгүй
- Ring structure эсвэл stockwork vein ажиглагдсан
- Pyrite, chalcopyrite, bornite, malachite илэрсэн
- Ойролцоо porphyry Cu-Au орд/илрэл байгаа

# 8. Go / No-Go decision хийх

## 8.1 Go decision
Дараах нөхцөл бүрдвэл Go гэж үзнэ.
- Target score 75-аас дээш
- 3 ба түүнээс дээш independent evidence давхцсан
- Lab assay эсвэл field mineralization баталгаатай
- Geological model нийцтэй
- Дараагийн шатны ажил хийх хангалттай үндэслэлтэй
## 8.2 Conditional Go
Дараах нөхцөлд Conditional Go гэж үзнэ.
- Target score 50–74
- Зарим evidence сайн боловч data gap байгаа
- QA/QC бүрэн биш
- Field verification шаардлагатай
- Дээжийн тоо хангалтгүй
## 8.3 No-Go
Дараах нөхцөлд No-Go гэж үзнэ.
- Target score 25-аас бага
- Evidence давхцал байхгүй
- Assay result сул
- Geological setting тохиромжгүй
- Remote sensing anomaly field evidence-ээр баталгаажаагүй
- CMCS болон ойролцоох ордтой холбоо сул
